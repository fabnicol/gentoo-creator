◆ test_cdrecord()

test_cdrecord ( )

Test whether cdrecord is functional.

Try to scan the SCSI bus and exit on error

Warning
There may be a rights issue with versions of cdrecord built from original source as they sometimes request elevated rights to perform burning with some hardware configurations.
In such an event, either run script with elevated rights or modify cdrecord rights using:
chown root cdrecord && chgrp bin cdrecord && chmod 04755 cdrecord
Return values
0on success otherwise exits -1 on failure.
